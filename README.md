# 🚀 TaskFlow - Team Management System

TaskFlow is a modern full-stack team and task management application built with the MERN stack. It helps teams manage projects, tasks, members, and workflow efficiently with a clean and responsive UI.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/taskflow.git
cd taskflow
```

---

## 2️⃣ Backend Setup

```bash
cd backend
npm install
```

### Create `.env` file inside backend

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

### Run Backend

```bash
npm run dev
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
```

### Run Frontend

```bash
npm start
```

# 🚀 Deployment

## Backend Deployment (Render)

1. Push project to GitHub
2. Create Web Service on Render
3. Add environment variables
4. Deploy backend

## Frontend Deployment (Vercel)

1. Import GitHub repository
2. Select frontend folder
3. Add production API URL
4. Deploy frontend
